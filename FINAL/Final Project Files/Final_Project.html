WEEK 4

~~~~~More Code.org: Making Games~~~~~


This week's assignment was to create two games of different types, once again using blocks of code in Code.org. The first, "Aliens on the Courts!", uses the WASD keys to control the character. Alien sprites are set to flee from the players position and the goal is to collide with each of the aliens until they're all gone.
The second, "Choose Your Own Dino Adventure", is a text-based choose your own adventure game using blocks of code to take the player to a different page depending on the button they choose to press. This one was really fun to make, although the first, more complex rendtion was erased so the version that you see here is a more condensed version.
